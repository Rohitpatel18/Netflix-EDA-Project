# Netflix Dataset Exploratory Data Analysis

## Project Overview

In this project, I performed Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset.
The goal of this analysis was to clean the dataset, explore different features, and understand patterns in Netflix's content library.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

## Data Cleaning

The dataset contained missing values which were handled before analysis:

* Missing values in columns such as director, cast, and country were handled
* The `date_added` column was converted into datetime format
* A new column `year_added` was created to analyze yearly trends

## Analysis Performed

The following analyses were performed during the project:

* Distribution of Movies vs TV Shows
* Top countries producing Netflix content
* Content rating distribution
* Release year distribution
* Trend of content added to Netflix over the years
* Most common genres on Netflix

## Key Insights

* Movies are more common on Netflix compared to TV Shows
* The United States contributes the highest number of titles
* Netflix started adding significantly more content after 2016
* TV-MA and TV-14 are the most common content ratings
* Drama and International TV Shows appear frequently on Netflix

## Conclusion

This analysis provides insights into Netflix's content distribution and highlights trends in content type, country contribution, ratings, and genre popularity.
